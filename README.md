# Audible-Project-EXCEL-PBI.
Project 1.
Audible Data cleaning.
The project aims to clean the audible data given using excel and power bi, create visualization using appropriate charts and graphs and find out relations and key factors within the data.
only excel and and power bi is used.
Data contains names,authors, narrators, time, release date, ratings and price.
Cleaned data using excel removed error columns, asssigned appropriate data type to each column used proper functiion to clean up names, cleaned author and narrator columns, created a minutes column from time column where time was mentioned in hours.
cleaned data was then importted into power bi where further  visualizatiion tools are used, Line chart is used to see audiobook release trends over years, two clustered bar charts are give one in which authors with most number of ratings are given and one bar chart shows number of ratings by narrators are shown. A donuut chart shows a breakdown of the catalog based on language, Another chart shows top authors by sum of price, Scatter chart shows average price and average ratings for top 1000 authors
from the visuals we can see that a total of 34 million minutes of audiobooks where played from 82.44k books consisting of 36 languages, average price of a book is 558.24rs, each book was rated an average of 2.42 times receiving an average of 0.74 rating
The top rated authors based on no of ratings are JK Rowling, Robin Sharma & Robertt kiyosaki, Most narrations where by anonymous narrators, Audiobook releases saw a continous upward spike from 2017 onwards reacing its peak in 2021 before starting to decline. English is the predominant language followed by german and spanish.
